- 前端：Vue.js

- 前端：现代浏览器

## 目录结构

```
/backend                 # 后端代码
    /app                 # Flask应用目录
        /models          # 数据模型
        /views           # 视图函数
        /static          # 静态文件
        /templates       # 模板文件
    /config.py           # 配置文件
    /run.py              # 应用启动文件
/frontend                # 前端代码
    /src                # Vue.js源代码目录
        /assets          # 资源文件
        /components      # 组件
        /views           # Vue视图
    /public             # 公共文件，如index.html
```

## 核心亮点

- 前后端分离，便于维护和扩展
- 灵活的定制化方案，适应不同业务需求
- 支持多种数据可视化方案，辅助决策分析
